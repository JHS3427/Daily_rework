1. sum
Select sum(population) from world - 월드 테이블에서 population컬럼 합계 출력 - where에 조건 걸어서 더 상세한 조건 걸기 가능.
2. count
select count(name) from world - 월드 테이블에서 name칸의 갯수 세서 출력.

3. Group by
select continent, count(name) from world where ~~~ group by continent - 월드 테이블에서 continent열의 고유 값들로 묶은 후 거기서 select를 진행함. 이때 where절이 존재하면 where절 걸고 남은 것들을 처리함.
*주의; Where절은 sum 같은 함수가 들어가면 안됨, having으로 넘겨야함

4. Having
select continent from world group by continent having sum(population)>100000000 - 월드 테이블에서 continent 열의 고유값들로 묶고, 묶은 것들에서 Having 절의 조건을 걸어서 일치하면 남김.
